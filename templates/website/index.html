    Welcome to your own personal sales assistant! This website is designed to help you track all aspects of your sales, from daily managing daily tasks to keeping track of your commission and incentive pay.

    This website is under ongoing construction and improvement, so be patient and feel free to suggest any new feature you would like to see implemented. Nothing is out of the realm of possbility. Be sure to keep an eye on the site news section of the site in order to stay up to date with the newly implemented features, as well as general site news.

    Currently this website's uses are fairly limited. After creating an account you can assign yourself to an existing company, then manage the incentives currently being offered by this company. There is also a task manager built into this website. You can input custom task information and set a date for which this task is due. After logging in you are brought to your user dashboard, where you will be shown upcoming tasks and incentives. The user dashboard is also where you will be able to access the task and incentive main menus.

    The most anticipated planned features are the ability to predict incentive payouts at the end of each month, as well a recurring email containing a list of tasks and upcoming incentives to you at whichever time interval you would prefer. The maojority of the work necessary to complete these features is simply a matter of the time it takes to implement, so as mentioned earlier please be patient and check back for updates!

    This is currently an open source project built with the main intention of helping me to learn full-stack web development, as well as teaching me how to build and implement custom web applications. Please feel free to contribute, or provide feedback. The image above links to the projects's github page.
